# Bus-System 🚌

A Java-based application for managing bus schedules, bookings, and user roles.

## 🚀 Features
* **Admin Dashboard:** Manage routes and schedules.
* **User Booking:** Book tickets and view history.
* **Role-Based Access:** Different views for Admins, Managers, and Users.

## 🛠️ Built With
* **Java** (JDK 21)
* **JavaFX** (for the UI/FXML files)
* **H2 Database** (the `.db` file in the repo)

## 📋 How to Run
1. Import this project into **Eclipse**.
2. Ensure you have the **JavaFX SDK** configured in your Build Path.
3. Run `Main.java` as a Java Application.
